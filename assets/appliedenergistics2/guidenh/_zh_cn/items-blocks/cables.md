# 线缆
AE线缆是连接AE各个组件或各个AE网络的物品。根据承载的[频道](../ae2-mechanics/channels.md)上限，AE线缆可以被简单地分为普通线缆与致密线缆，普通线缆能承载最多8个频道，致密线缆能承载最多32个频道。
# 普通线缆
常用的普通线缆有<ItemLink id="appliedenergistics2:item.ItemMultiPart:16" showIcon="left" />与<ItemLink id="appliedenergistics2:item.ItemMultiPart:36" showIcon="left"/>，需要注意的是玻璃线缆占用的渲染资源更高，极端情况下可能影响游戏帧数。

# 致密性线缆
致密线缆除了有更高的频道上限，也有更高的[频道路由](../ae2-mechanics/channels.md#频道路由)优先级。

# 智能线缆
智能线缆会通过线缆上的纹路明暗来显示线缆当前承载的频道数量<Spoiler>（纹路真的很难看清！！）</Spoiler>。但由于智能线缆的造价与更高的渲染占用，并不推荐大量使用。
